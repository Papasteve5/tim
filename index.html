Cloud Computing

Einfach erklärt ist Cloud Computing die Bereitstellung von Computer-Services über das Internet (die Cloud)

Vorteile

Keine Softwarekosten

Statt teure Systeme und Geräte zu kaufen, können Kosten gesenkt werden, indem die Ressourcen der Cloud Dienstleister genutzt werden.

Effizientes Teamwork

Wenn mehrere an einem Projekt arbeiten, kann die Cloud genutzt werden, um ganz einfach allen Zugriffe zu verteilen oder Daten und Tools auszutauschen.

Skalierbarkeit

Man ist sehr flexibel bei Änderungen und man muss nicht selber Upgrades durchführen, das übernimmt der Cloud Anbieter.

The Big 3

Die 3 Haupt Cloud Services sind: SaaS, PaaS und IaaS

Software as a Service (SaaS)

Voll entwickelte Cloud Applikation SaaS werden auf Remote-Computern in der Cloud ausgeführt. Bekannte Beispiele für SaaS sind zum Beispiel: Online Emails (Gmail), Office 365
